# AI Review Packet

This packet is meant to be shared with other AI models that do not have direct access to the full codebase.

Contents:
- `review_prompt.txt`: paste-ready review prompt
- `codex_edge_case_detection.md`: main memo to critique
- `01-system-map.md`: system structure overview
- `03-backend-analysis.md`: backend architecture and implementation analysis
- `05-reliability-and-testing.md`: reliability and test findings
- `06-prioritized-roadmap.md`: broader code-grounded roadmap
- `evidence_appendix.md`: direct code excerpts supporting key claims in the memo

Recommended use:
1. Give the model `review_prompt.txt`
2. Attach all markdown files in this folder
3. Ask the model to stress-test the memo, not summarize it
